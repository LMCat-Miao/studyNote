# 面试最经典题

## 题目：

```
console.log("1");


setTimeout(()=>{
    console.log("2");


    Promise.resolve()
    .then(()=>{
        console.log("3");
    })

},0);



Promise.resolve()
.then(()=>{

    console.log("4");

})


console.log("5");
```

------

我们分析。

------

同步：

```
1

5
```

------

微任务：

Promise.then:

```
4
```

------

宏任务：

setTimeout:

输出：

```
2
```

里面产生新的微任务：

```
3
```

所以：

最终：

```
1
5
4
2
3
```

------

# async/await 是面试重点

看：

```
async function test(){

    console.log(1);

    await Promise.resolve();

    console.log(2);

}


test();


console.log(3);
```

很多人错。

认为：

```
1
2
3
```

错误。

------

执行：

调用test：

进入函数：

同步：

```
console.log(1)
```

输出：

```
1
```

遇到：

```
await
```

暂停。

后面的：

```
console.log(2)
```

进入微任务。

继续主线程：

```
console.log(3)
```

输出：

```
3
```

微任务：

输出：

```
2
```

最终：

```
1
3
2
```
